## Useful Git Commands

Clone repository

```bash
git clone <repository-url>
```

Check status

```bash
git status
```

Add files

```bash
git add .
```

Commit changes

```bash
git commit -m "First commit"
```

Push to GitHub

```bash
git push origin main
```
